[파트 1: 데이터의 이해 및 디지털 표현]
1. 다음 중 '데이터'의 정의로 가장 적절한 것은? ① 인간의 직관적인 느낌 ② 숫자, 문자, 이미지, 소리 등으로 표현된 사실이나 기록 ③ 컴퓨터 하드웨어의 물리적 부품 ④ 정답이 정해져 있는 수학 공식 ⑤ 아무런 의미가 없는 단순한 소음
2. 아날로그와 디지털에 대한 설명으로 옳은 것은? ① 아날로그는 0과 1로만 데이터를 표현한다. ② 디지털은 끊어짐 없이 연속적인 값으로 데이터를 표현한다. ③ 수은식 온도계는 디지털 방식의 대표적인 예이다. ④ 디지털은 띄엄띄엄 떨어진 값을 사용하여 데이터를 표현한다. ⑤ 자연 현상에서 발생하는 물리량은 대개 디지털 형태이다.
3. 디지털 방식이 아날로그 방식에 비해 가지는 장점은? ① 정보의 손실이 전혀 발생하지 않는다. ② 노이즈(잡음)에 상대적으로 둔감하고 안정적이다. ③ 별도의 변환 과정 없이 컴퓨터가 바로 이해한다. ④ 데이터의 양이 아날로그보다 항상 적다. ⑤ 기온 변화를 더 정밀하게 무한대로 표현할 수 있다.
4. 디지털 데이터를 표현하는 가장 기본 단위는? ① 바이트(Byte) ② 비트(Bit) ③ 화소(Pixel) ④ 프레임(Frame) ⑤ 노드(Node)
5. 3개의 비트를 사용하여 표현할 수 있는 정보의 가짓수는 총 몇 개인가? ① 3개 ② 4개 ③ 6개 ④ 8개 ⑤ 16개
6. 아스키(ASCII) 코드에 대한 설명으로 옳지 않은 것은? ① 알파벳 대소문자와 숫자 등을 표현하는 표준 코드이다. ② 7개의 비트를 사용하여 문자 한 개를 표현한다. ③ 총 128개의 문자를 표현할 수 있다. ④ 한글과 한자 등 다양한 국가의 문자를 표현하기에 충분하다. ⑤ 특수 문자를 포함하고 있다.
7. 전 세계의 모든 문자를 디지털로 표현하기 위해 설계된 표준 코드는? ① 아스키 코드 ② 모스 부호 ③ 유니코드 ④ QR 코드 ⑤ 16진수 코드
8. 유니코드의 특징으로 옳은 것을 모두 고르면? ① 7비트만 사용하여 모든 데이터를 저장한다. ② 현대 한글 음절 11,172자를 포함한다. ③ 이모티콘이나 특수 문자는 표현할 수 없다. ④ 알파벳 전용으로 개발되었다. ⑤ 데이터 용량이 아스키 코드보다 항상 작다.
9. 디지털 이미지를 이루는 가장 작은 점을 무엇이라 하는가? ① 비트 ② 화소(Pixel) ③ 해상도 ④ 프레임 ⑤ 샘플
10. 이미지의 세밀한 정도를 나타내며, 가로와 세로의 화소 수 곱으로 표현하는 것은? ① 채도 ② 명도 ③ 해상도 ④ 비트 레이트 ⑤ FPS
11. 한 개의 화소에 2개의 비트를 사용하면 표현할 수 있는 색상의 수는? ① 2가지 ② 4가지 ③ 8가지 ④ 16가지 ⑤ 256가지
12. 디지털 이미지에서 빛의 3원색(RGB)을 표현하기 위해 주로 사용하는 비트 수는? ① 1비트 ② 8비트 ③ 16비트 ④ 24비트 ⑤ 64비트
13. 소리의 디지털 변환 과정 순서가 올바르게 나열된 것은? ① 양자화 → 표본화 → 부호화 ② 표본화 → 양자화 → 부호화 ③ 부호화 → 표본화 → 양자화 ④ 표본화 → 부호화 → 양자화 ⑤ 양자화 → 부호화 → 표본화
14. 소리 변환 과정 중 '아날로그 소리 파형을 일정한 시간 간격으로 추출하는 단계'는? ① 양자화 ② 부호화 ③ 표본화 ④ 근사화 ⑤ 시각화
15. 양자화 과정에서 표현하는 정수의 범위가 커질 때 나타나는 결과는? ① 음질이 나빠진다. ② 데이터 용량이 줄어든다. ③ 원음에 더 가까운 소리를 표현할 수 있다. ④ 소리의 높낮이가 변한다. ⑤ 변환 속도가 느려지지만 음질은 동일하다.
16. 동영상의 디지털 표현 방식에 대한 설명으로 옳은 것은? ① 동영상은 연속된 소리 데이터의 집합이다. ② 정지된 이미지 데이터를 빠르게 연속적으로 보여주는 방식이다. ③ 프레임 수가 적을수록 움직임이 부드럽다. ④ 초당 프레임 수(FPS)가 높으면 용량이 줄어든다. ⑤ 소리와 이미지는 별개의 원리로 동영상에 저장된다.
17. 1초에 연속적으로 재생하는 프레임의 수를 나타내는 단위는? ① Bit ② Pixel ③ FPS ④ RGB ⑤ ASCII
18. 영화에서 많이 사용하는 방식인 '24FPS'는 1초에 몇 장의 이미지를 재생하는가? ① 12장 ② 24장 ③ 30장 ④ 60장 ⑤ 128장
19. 디지털 전환(Digital Transformation)의 사례로 적절하지 않은 것은? ① 수기 장부 대신 컴퓨터 데이터베이스 사용 ② 영화관 방문 대신 실시간 영상 추천 서비스 이용 ③ 종이 신문만 제작하여 배달하는 방식 ④ 스마트폰 앱을 통한 비대면 주문 서비스 ⑤ 종이 서류를 디지털 파일로 변환하여 관리
20. 조선왕조실록을 디지털화했을 때의 장점으로 소스에서 언급된 것은? ① 실록의 물리적 무게를 줄일 수 있다. ② 곰팡이나 세균 번식을 막기 위한 포쇄 작업이 더 필요해진다. ③ 누구나 인터넷을 통해 내용을 쉽게 검색하고 연구할 수 있다. ④ 종이 기록보다 역사적 가치가 낮아진다. ⑤ 디지털 데이터는 변조가 불가능하여 보안 설정이 필요 없다.

[파트 2: 데이터 수집, 관리, 구조화 및 분석]
21. 다음 중 데이터 수집의 중요성에 대한 설명으로 가장 적절한 것은? ① 느낌이나 직관에만 의존하여 의사 결정을 내리기 위해서이다. ② 합리적이고 객관적인 방향으로 문제를 해결하는 데 도움을 준다. ③ 데이터가 많을수록 항상 문제 해결 속도가 빨라지기 때문이다. ④ 디지털 데이터는 무조건 신뢰할 수 있기 때문이다. ⑤ 데이터 수집은 전문가만 할 수 있는 영역이기 때문이다.
22. 사람의 눈으로 보거나 센서를 통해 데이터를 직접 측정하여 수집하는 방법은? ① 설문 조사 ② 관찰 ③ 웹 검색 ④ 공공 데이터 활용 ⑤ 메타 데이터 분석
23. 공공 기관이 보유하여 공개하는 데이터를 수집하는 방법의 예는? ① 친구에게 질문지를 돌려 간식 선호도 조사하기 ② 기상청 사이트에서 과거 기온 데이터 내려받기 ③ 도서관에서 책의 내용을 직접 손으로 베껴 쓰기 ④ 특정 웹 사이트의 정보를 자동으로 크롤링하기 ⑤ 식물원의 꽃이 피는 시기를 매일 가서 기록하기
24. 웹 검색 시 '정확히 일치하는 결과'를 찾기 위해 검색어에 사용하는 기호는? ① + ② - ③ | ④ " " ⑤ *
25. 데이터 수집 시 주의할 점으로 옳지 않은 것은? ① 문제 해결 목적에 적합한 데이터인지 확인한다. ② 데이터의 출처가 명확하고 신뢰할 수 있는지 확인한다. ③ 수집한 데이터가 최신의 것인지 검토한다. ④ 데이터의 양은 적을수록 분석이 빠르므로 최소화한다. ⑤ 데이터가 특정 방향으로 치우치지 않았는지 확인한다.
26. 수집한 데이터를 폴더 구조로 나누어 관리할 때의 장점은? ① 데이터의 용량을 획기적으로 줄여준다. ② 필요한 데이터를 더 쉽게 찾아 활용할 수 있다. ③ 모든 데이터가 자동으로 최신 상태를 유지한다. ④ 데이터의 오류를 스스로 수정해 준다. ⑤ 폴더 이름만으로 데이터의 세부 내용을 모두 알 수 있다.
27. 데이터에 대한 정보(속성, 크기, 생성 일자 등)를 설명하는 데이터는? ① 원시 데이터 ② 메타 데이터 ③ 구조화 데이터 ④ 시각화 데이터 ⑤ 가공 데이터
28. 데이터 구조화의 목적으로 옳은 것은? ① 데이터의 크기를 무조건 크게 만들기 위해 ② 데이터 간의 관계를 쉽게 파악하고 이해하기 위해 ③ 아날로그 데이터를 다시 만들기 위해 ④ 데이터 수집 시간을 늘리기 위해 ⑤ 데이터를 복잡하게 만들어 보안을 강화하기 위해
29. 데이터를 나열하여 관리하며, 한 가지 속성을 기준으로 정렬할 때 유용한 구조는? ① 트리 ② 그래프 ③ 리스트 ④ 표 ⑤ 메타 데이터
30. 데이터를 행과 열을 기준으로 나열하며 여러 속성을 정리하기에 좋은 구조는? ① 리스트 ② 표(테이블) ③ 트리 ④ 그래프 ⑤ 비트
31. 부모 노드와 자식 노드 사이의 상하 관계나 포함 관계를 표현하는 구조는? ① 그래프 ② 트리 ③ 리스트 ④ 비트맵 ⑤ 픽셀
32. 사람 사이의 관계, 도로망, 네트워크 등 복잡한 관계를 점과 선으로 표현하는 구조는? ① 트리 ② 리스트 ③ 표 ④ 그래프 ⑤ 16진수
33. 데이터 분석의 일반적인 3단계 과정이 올바른 것은? ① 데이터 수집 → 데이터 시각화 → 데이터 전처리 ② 데이터 전처리 → 데이터 수집 → 데이터 시각화 ③ 데이터 수집 → 데이터 전처리 → 데이터 시각화 ④ 데이터 시각화 → 데이터 전처리 → 데이터 수집 ⑤ 데이터 전처리 → 데이터 시각화 → 데이터 수집
34. '수집한 데이터를 분석하기에 적합한 형태로 다듬고 가공하는 과정'을 무엇이라 하는가? ① 데이터 구조화 ② 데이터 전처리 ③ 데이터 시각화 ④ 데이터 부호화 ⑤ 데이터 양자화
35. 데이터 분석 과정 중 '이상치(outlier)'에 대한 설명으로 옳은 것은? ① 데이터 수집 시 누락된 값을 의미한다. ② 일반적인 데이터 범위에 비해 아주 작거나 큰 값을 의미한다. ③ 데이터 분석 결과를 항상 더 정확하게 만들어준다. ④ 이상치는 무조건 삭제해야만 하며 대체할 수 없다. ⑤ 모든 데이터는 이상치를 포함하고 있어야 분석이 가능하다.
36. 데이터 수집 과정에서 누락되거나 측정되지 않은 데이터를 뜻하는 용어는? ① 이상치 ② 결측치(missing value) ③ 잡음(noise) ④ 노드 ⑤ 속성
37. 숫자로 이루어진 데이터의 의미를 그래프 등을 활용해 파악하기 쉽게 표현하는 방법은? ① 부호화 ② 구조화 ③ 시각화 ④ 크롤링 ⑤ 표본화
38. 시간 간격에 따른 데이터의 변화를 보여주기에 가장 적합한 시각화 방법은? ① 막대 그래프 ② 원그래프 ③ 선그래프 ④ 산점도 ⑤ 트리
39. 전체에 대한 각 부분의 데이터가 차지하는 비율(백분율)을 나타내는 데 유용한 그래프는? ① 선그래프 ② 원그래프 ③ 막대 그래프 ④ 산점도 ⑤ 리스트
